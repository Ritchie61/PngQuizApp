# QuizApp

A simple React Native quiz app that presents multiple-choice questions to the user.

## Features

- Multiple-choice questions
- User feedback for correct and incorrect answers
- Score tracking

## Getting Started

### Prerequisites

- Node.js installed on your machine
- React Native CLI installed
- Android Studio or Xcode (for running the app)

### Installation

1. Clone the repository:
   git clone https://github.com/yourusername/QuizApp.git

2. Navigate to the project directory:
   cd QuizApp

3. Install dependencies:
   npm install

### Running the App

- For Android:
  npx react-native run-android

- For iOS:
  npx react-native run-ios

## API Endpoints (if applicable)

- `GET /api/quizzes` - Retrieve all quizzes
- `POST /api/quizzes` - Add a new quiz
- `GET /api/quizzes/:id` - Retrieve a specific quiz by ID
